# 📚 Catálogo de Retos - LeetCode Local

Este laboratorio contiene **más de 140 retos y componentes interactivos** diseñados para cubrir el ciclo completo de vida del desarrollo de software con Java 21 y Spring Boot 3.

## 🟢 Nivel 00: Iniciación al Testing (AAA)
*Aprende a escribir pruebas unitarias para código existente.*
- **Calculadora Financiera**: Interés compuesto.
- **Calculadora Simple**: Sumas básicas.
- **Contador Elementos**: Manejo de nulos en listas.
- **Transformador Texto**: Limpieza de Strings.
- **Validador Rango**: Lógica booleana básica.

## 🟢 Nivel 01: Algoritmia y Lógica Pura
- **Arrays**: Busqueda Binaria, Contar Islas, Encontrar Duplicado, Invertir Lista Enlazada, Máximo Subarreglo (Kadane), Mover Ceros, Número Faltante, Producto Excepto Propio, Rotar Array, Suma Diagonal, Two Sum.
- **Recursión**: Factorial, Fibonacci, Suma de Dígitos.
- **Strings**: Compresión de Cadenas, Contar Vocales, Detector de Anagramas, Invertir Palabras, Invertir Vocales, Palíndromo, Primer Carácter Único, Subcadena sin Repetir, Validar Paréntesis.
- **Trees**: Invertir Árbol Binario.

## 🔵 Nivel 02: Java Funcional (Streams & Lambdas)
- **Transformaciones**: Filtro E-commerce, Calculadora Recetas, Filtro Usuarios, Procesador Pedidos, Unión de Nombres, Filtro de Nombres Únicos.
- **Agregaciones**: Estadísticas Números, Estadísticas Ventas, Frecuencia de Palabras.
- **Clasificación**: Agrupador Jerárquico, Agrupador Monto Ventas, Agrupador Rangos Edad, Filtro de Ciudades, Buscador de Empleados, Buscador de Iniciales, Filtro de Números Primos.
- **Utilidades**: Calculadora de IVA.

## 🟡 Nivel 03: Lógica de Negocio (Spring Services)
- **Fintech**: Procesador de Pagos, Validador de Fraude, Calculadora de Lealtad, Calculadora de Suscripción, Conversor de Monedas, Generador de Referencias.
- **Logística**: Calculadora de Envíos Internacionales.
- **Juegos**: Motor Roguelike, Movimientos de Ajedrez.
- **Orquestación**: Orquestador de Pedidos.
- **Validación**: Validador de Email, Validador de Inscripciones, Validador de Seguridad de Contraseñas.

## 🟠 Nivel 04: Persistencia Relacional (Spring Data JPA)
- **Entidades**: Cliente, Coche, Curso, Empresa, Estudiante, Item, Jugador, Pedido, Producto, Proyecto, Tarea, Ticket, Transacción, Usuario.
- **Consultas**: Búsquedas por atributos, TopN (Rankings), Rangos de fecha, Filtros complejos, Relaciones @ManyToOne, Atributos @Embedded.

## 🟣 Niveles Superiores: Arquitectura e Infraestructura
- **Nivel 05-06**: Controladores REST profesionales, Patrones DTO, Mappers, Manejo Global de Excepciones.
- **Nivel 07**: Programación Orientada a Aspectos (AOP) - Auditoría automática y validación de argumentos.
- **Nivel 12**: Seguridad Industrial (JWT, Roles, Cifrado BCrypt).
- **Nivel 13-14**: Optimización con Redis Cache (Manual y Declarativo) y Mensajería con RabbitMQ.
- **Nivel 15-17**: Métricas con Actuator/Prometheus, Migraciones evolutivas con Flyway.
- **Nivel 18-20**: Tareas Programadas (Scheduler), Integración Externa (WebClient), Notificaciones por Email.
- **Nivel 21-24**: Documentación Visual con Mermaid, APIs con GraphQL, Resiliencia (Circuit Breaker) y Trazabilidad (Zipkin).
- **Nivel 25-32**: Spring Batch, Auditoría con Hibernate Envers, Pruebas Parametrizadas, Orquestación Global y Lógica Reactiva.
